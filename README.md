# Global Job Market Mapper
### Problem Statement
### Description
A career-planning tool, for people who have finished university or looking for a career change in the UK, that identifies which technical skills are currently in demand and what they pay. The job raw job data shall be obtained from the Adzuna API<br>
### Features
Scrape job boards to get raw job descriptions<br>
Obatin structured job data and salary estimates<br>
Allow user to input skill(s) and get a rough salary estimate<br>
Allow user to see what skill(s) are most valuable in a field<br>
Allow user to see what skill(s) are most valuable in general<br>
Return Number of job postings requiring specific skill(s)<br>
Return Average or median salary (if available)<br>
Return Top 10 most in-demand skills
Realtionships between Jobs, Skills and Companies shown in tables<br>
